# Build Manual — Digital Signage (Flutter TV)

## پیش‌ نیازها
- Flutter SDK (ورژن 3.x)
- Android Studio + SDK Tools
- JDK 17 (یا سازگار)
- دستگاه Android یا TV با USB debugging فعال

---

## build برای دیباگ (اجرای سریع)
flutter run

## build ریلیز (APK)
flutter build apk --release
